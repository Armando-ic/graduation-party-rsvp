# Resume Entry — Graduation Party RSVP Project

## Project Section Entry

**Graduation Party RSVP & Photo Sharing Platform**
*Personal Project | Firebase, JavaScript, GCP, HTML/CSS*

- Built and deployed a serverless web application for managing 30-100 guest RSVPs, real-time photo sharing, and event photo gallery using vanilla JavaScript and Firebase (Firestore, Cloud Storage, Hosting, Auth)
- Implemented Firebase App Check with reCAPTCHA Enterprise and hardened Firestore/Storage security rules with field-level validation, admin UID locking, and Content Security Policy headers
- Designed a single-page tabbed architecture with lazy module initialization, client-side image compression, and real-time Firestore listeners for live dashboard updates
- Created an admin dashboard with real-time analytics, CSV export, QR code generation, and media management capabilities

**Live:** gmu-graduation-party-rsvp.web.app | **GitHub:** github.com/Armando-ic/graduation-party-rsvp
